Tag Definition Examples
<!DOCTYPE html> Declaration that informs the browser of the document type. No examples*
<html lang="en"> The main element of an HTML page. This page is set to “en” for English. This tag helps websites such as Google Translate know the default language for this page is English.
<meta charset="UTF-8"> Specifies character encoding. No examples*
<title> Sets the title shown in the browser tab. Assignment 3 (check the browser tab).
<style> Used to define internal CSS. You can change the style of certain elements.
<link rel="stylesheet" href="styles.css"> Used to link external CSS files. The homepage and this page point to a common style.css file that controls the look of these pages.
<link rel="icon" type="image/x-icon" href="favicon.ico"> Used to link the favicon for the page. <-- This is a favicon (check the browser tab).
